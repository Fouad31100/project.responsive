Logiciel CLIPBOARD

Clipboard Plus est un logiciel permettant de gérer votre presse-papier. Aussi simple que complet, l'utilitaire est capable de gérer plusieurs entrées pour vous permettre de les retrouver facilement. Ainsi, Clipboard Plus garde en mémoire les cinq derniers éléments que vous avez pu copier dans le presse-papier de Windows. Affichés au sein d'une liste du plus récent au plus ancien, Clipboard Plus vous permet de récupérer un élément copié à l'aide d'un simple clip. Le contenu prend alors la première place de la liste, faisant reculer les autres d'une place. A noter que l'utilitaire est capable d'ouvrir les URL directement simplement en cliquant dessus.

Un clic droit dans l'interface de Clipboard Plus vous permet d'accéder aux options. Il est alors possible de nettoyer toutes les entrées, de les sauvegarder, ou activer la fonction Password Mode qui permettra de masquer les suites de caractères détectées comme étant des mots de passe.
